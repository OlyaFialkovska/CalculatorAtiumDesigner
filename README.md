🧮 Калькулятор на дисплеї OLED I2C + STM8   

Автор: Фіалковська Ольга
Платформа: STM8
Дисплей: OLED SSD1306 (інтерфейс I2C)

📖 Опис проєкту
Цей проєкт — електронний калькулятор, побудований на мікроконтролері STM8.
Він виконує базові та розширені арифметичні операції, відображаючи результати на OLED-дисплеї SSD1306 через інтерфейс I2C.
Керування здійснюється за допомогою 24-кнопкової клавіатури, а активний зумер забезпечує звукове підтвердження натискань.

Калькулятор використовує вхідні регістри 74HC165 для зчитування сигналів з клавіатури, а конденсатори 100 нФ розміщено поруч із регістрами для фільтрації шумів.

⚙️ Основні можливості
➕➖✖️➗ Виконання базових арифметичних операцій
🧠 Додаткові функції: √x, x^y, дужки (), десяткові числа, зміна знаку +/-
🔊 Звукове підтвердження натискання кнопок
🕒 Автовимкнення дисплея після періоду неактивності
💾 Робота через зсувні регістри 74HC165
🔴 Червоний світлодіод для індикації стану

| Компонент          | Модель / Значення  | Призначення                      |

| **Мікроконтролер** | STM8               | Основний обчислювальний блок     |

🧠 Логіка роботи
Натискання клавіш зчитується через регістри 74HC165.
Дані передаються у мікроконтролер STM8.
Після обробки результат відображається на OLED SSD1306 через I2C.
Зумер видає короткий сигнал при натисканні.
Світлодіод індикує активний стан пристрою.
